# Совместимость

Проект совместим с *node.js 18*

# Локальная работа с проектом

- client

В .env файле указать *NEXTAUTH_SECRET*

Выполнить команду:

```
npm run dev
```

- server

В .env файле указать *JWT_SECRET_KEY*, *JWT_REFRESH_TOKEN* и *DATABASE_URL* (указан в .env.example)

Выполнить команду:

```
npm run start:dev
```
